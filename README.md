# iw8-nowatermark

NOP patch to remove the IW8 dev watermark overlay.

## Installation

1. Download `version.dll` from the [Releases](../../releases) page.
2. Navigate to your Call of Duty: Modern Warfare mod tools installation folder, for example:
   ```
   C:\Program Files (x86)\Steam\steamapps\common\Call of Duty Modern Warfare\
   ```
3. Place `version.dll` in the root of that folder (the same directory that contains `ModernWarfare.exe`).
4. Launch the game or mod tools as usual — the dev watermark will no longer appear.
